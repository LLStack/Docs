# 介绍

LiteSpeed 即 LiteSpeed Web Server（简称 LSWS），是一款企业级商用 Web 服务软件，可以完美的 Apache HTTPD 兼容体验，.htaccess 规则可以直接兼容而不像 Nginx 需要重写，并兼容常用的mod扩展。得益于基于事件的架构优势，静态内容比Apache Httpd 快 5 倍 ，动态内容更是快 40 倍，HTTPS 访问快 3 倍并可以应用硬件加速器。

# LiteSpeed 特性

# Apache 插入式替换

直接读取 Apache HTTPD 配置文件，支持 .htaccess，WAF 等功能，不需要停机

# HTTP / 2 /3支持

HTTP / 2，HTTP / 3可以更快地将内容提供给消费者设备

# QUIC支持

使用QUIC减少连接建立时间并改善拥塞控制

# 零停机时间

只有LiteSpeed Web Server允许您更新Web服务器软件而无需停机

# 缓解DDoS攻击

LiteSpeed Web Server 可缓解拒绝服务攻击

# 事件驱动

LSWS是事件驱动的，Apache是基于进程的。

# 细节特点

# 基本功能

  • HTTP / 2
  • 符合HTTP 1.0 / 1.1
  • 脚本语言：PHP，Perl，Ruby，Python，Java等......
  • 服务器API：LiteSpeed SAPI（LSAPI），CGI，FCGI，AJPv13，代理
  • HTTPS
  • IPv4和IPv6支持
  • 无限的IP和基于名称的虚拟主机
  • 无限制的并发连接
  • Brotli压缩
  • GZIP压缩
  • 支持SPDY / 2,3和3.1
  • WebSocket代理
  • 在Linux，FreeBSD，MacOSX，Solaris上运行

# 性能和可扩展性

  • 事件驱动的架构
  • 成千上万的并发连接
  • 微小的内存占用
  • 发送文件（）
  • 异步I / O支持
  • LiteSpeed的独家服务器API（LSAPI）可提供最佳的动态内容性能
  • 无与伦比的PHP suEXEC性能，适用于共享主机
  • 三种不同的PHP suEXEC设置，满足不同的需求和能力
  • 静态内容比Apache快5倍
  • 动态内容比Apache快40倍
  • HTTPS比Apache快3倍
  • OCSP装订
  • 内置页面缓存
  • 通过ESI支持涡轮增压Magento和其他Web应用程序
  • 高级.htaccess缓存消除了与Apache中使用.htaccess相关的性能瓶颈
  • CloudLinux LVE兼容性
  • 提高Web应用程序的可伸缩性
  • 服务器和Web应用程序的分离允许缓冲请求和响应，从而更好地处理多个连接
  • 及时配置削减重启时间
  • CPU亲和性允许更多地使用CPU缓存

# 便于使用

  • 控制面板插件（cPanel，DirectAdmin和Plesk）
  • 由cPanel正式支持
  • 直观的WebAdmin控制台GUI
  • 虚拟主机模板，便于配置
  • 实时服务监控

# 安全

  • wp-login.php的暴力保护
  • 内置防DDoS连接和带宽限制
  • mod_security兼容性
  • 与Atomic Secured Linux和Atomicorp Realtime ModSecurity规则的兼容性
  • 反SSL BEAST和重新协商攻击功能
  • 严格的HTTP请求验证
  • 拒绝缓冲区溢出尝试
  • PHP / Ruby / CGI / FCGI / LSAPI suEXEC
  • CloudLinux CageFS兼容性
  • 外部应用程序和完整的服务器chroot能力

# 高可用性

  • 应用更改，零停机时间
  • 零停机服务器软件升级
  • 立即从服务故障中恢复

# Apache兼容性

  • 直接读取Apache的配置文件
  • 替换cPanel，Plesk，DirectAdmin，Interworx等的Apache替代品
  • 完整的.htaccess支持
  • 与Apache核心模块兼容

# CloudLinux兼容性

  • 轻量级虚拟环境（LVE）
  • CageFS
  • PHP选择器
  • Node.js选择器
  • Python和Ruby Selector
  • 用户空间中的检查点/还原（CRIU）
